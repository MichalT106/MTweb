Project overview

CarInsight is a mobile app that helps drivers manage vehicles and responsibilities in one place.

Users can add multiple vehicles, track key events (technical inspections, insurance, highway vignettes, oil changes), and receive reminders ahead of time via notifications and a clear calendar view.
